使用方法

  1. 使用 ./gradlew buildPlugin 生成插件包，并在 GoLand 中通过 Install Plugin from Disk 安装
  2. 重启 IDE，在左侧工具窗口打开 Go Module Finder
  3. 输入模块名称（可选：作者、版本号），选择来源并点击搜索
  4. 在列表中选择模块，选择版本后点击下载
配置入口：Preferences → Tools → Go Module Finder
